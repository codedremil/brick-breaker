# brick-breaker
Ce jeu est une traduction du tutorial de Google pour apprendre Dart et Flutter.

Chaque répertoire est une version incrémentale du jeu.

Les étapes sont documentées dans les fichiers du répertoire ./doc.

Pour compiler les versions du jeu :

```
$ git clone https://github.com/codedremil/brick-breaker.git
```

Puis vous vous déplacez dans un répertoire (v1, v2,....), et vous exécutez les commandes suivantes :

```
$ flutter create .
$ flutter pub get
$ flutter run
```

Pour obtenir un fichier APK, installable sur votre téléphone, exécutez la commande suivante :

```
$ flutter build apk
```
